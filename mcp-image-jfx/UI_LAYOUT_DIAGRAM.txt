LEGEND:
────────
● Blue Circle    = Pending
● Orange Circle  = Running
● Green Circle   = Completed
● Red Circle     = Error

[Button]         = Clickable button
[Text Field]     = Editable text input
[Dropdown ▼]     = ComboBox/DropDown
▼ Category       = Expandable/Collapsible section
* Required       = Required parameter


INTERACTION FLOWS:
──────────────────

1. BUILDING A PIPELINE:
   a. Search for tool in left panel
   b. Drag tool from collection
   c. Drop in center pipeline area
   d. Select tool in pipeline
   e. Edit parameters in right panel
   f. Repeat for each tool

2. EXECUTING A PIPELINE:
   a. Click [Launch MCP Server] if not connected
   b. Wait for green status indicator
   c. Click [Play] button
   d. Watch status indicators change color
   e. Monitor progress bar in status bar
   f. Check final status message

3. PARAMETER CHAINING:
   a. Set output_key on first tool (e.g., "original")
   b. Use result_key on next tool to reference "original"
   c. Continue chaining through pipeline
   d. Final tool can save with output_path

4. WORKFLOW PERSISTENCE:
   a. Build desired pipeline
   b. Click [Save Workflow]
   c. Choose location and filename
   d. Later: Click [Load Workflow]
   e. Select saved file
   f. Pipeline restored with all parameters
